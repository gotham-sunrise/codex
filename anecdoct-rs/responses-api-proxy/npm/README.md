# @lambogenius/anecdoct-responses-api-proxy

<p align="center"><code>npm i -g @lambogenius/anecdoct-responses-api-proxy</code> to install <code>anecdoct-responses-api-proxy</code></p>

This package distributes the prebuilt [Anecdoct Responses API proxy binary](https://github.com/lambogenius/anecdoct/tree/main/anecdoct-rs/responses-api-proxy) for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/anecdoct-responses-api-proxy.js --help
```

Refer to [`anecdoct-rs/responses-api-proxy/README.md`](https://github.com/lambogenius/anecdoct/blob/main/anecdoct-rs/responses-api-proxy/README.md) for detailed documentation.
